--------------------------------------------------
Intel(R) MPI Benchmarks PREVIEW MT part
README
--------------------------------------------------

Intel(R) MPI Benchmarks (MT part) provides a set of elementary benchmarks that conform
to MPI-1 standard. All the benchmarks are designed to be run in hybrid
OpenMP-MPI mutithreaded mode. 

This is unofficial snapshot for preview only, might be changed massively in
future and is not for getting any official benchmark data.

This is a preview version only, do not share it!
